# A Memster Mod DEMO (V-SLICE PORT)

Official V-Slice basegame port.

In this mod, you and the players are introduced to a goofy storyline featuring two original characters: **Memster** and **Ink**. The demo is made to be fun, light-hearted, and a little chaotic.

## About Memster

Memster is lovable, mischievous, and always trying to make things fun. He brings a comedic twist to the game and keeps the experience playful.

## Usage

Use this with **V-Slice 0.3.X**.
